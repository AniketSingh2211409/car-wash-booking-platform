# Car Wash Booking Platform 🚗

A full-stack web application for managing car wash bookings and business operations.  
This project includes a customer-facing interface as well as an admin dashboard for managing businesses and platform activity.

---

## Features

- Browse nearby car wash centers  
- Book services by selecting date and time  
- Admin dashboard to manage businesses  
- Approve or update business details  
- Basic financial overview and stats  

---

## Tech Stack

- Frontend: React + TypeScript  
- Backend: Node.js + Express  
- Styling: Tailwind CSS  
- Charts: Recharts  

---

## Project Structure

- `/src` → Frontend (React app)  
- `/api` → Backend routes  
- `/server.ts` → Express server setup
